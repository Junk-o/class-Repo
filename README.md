# Hello World Repo

This repository is for learning how to create a Hello World project.

## About the Project

This project is a simple C++ program that prints "Hello World" to the console and exits successfully.

## How to Build and Run on Ubuntu (Command Line)

1. Install the C++ compiler:

        sudo apt install g++

2. Compile the program:

        g++ main.cpp -o hello

3. Run the program:

        ./hello

## How to Build on Windows (Visual Studio)

1. Open the project in Visual Studio
2. Select Build > Build Solution (or press Ctrl+Shift+B)
3. Press Ctrl+F5 to run without debugging

## Expected Output

    Hello World
